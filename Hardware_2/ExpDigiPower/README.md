[MicroMultimeter - ExpDigiPower](https://github.com/KitSprout/MicroMultimeter)
========
* Author  : [Hom](http://about.me/Hom)
* Version : v1.0
* Update  : 2015/05/18

Description
========
將 MizroMultimeter 的擴充，透過 DigiPot 調整穩壓 IC 的回授來實現數位電源，ExpID = [0, 1]。
